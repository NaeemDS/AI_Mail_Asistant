# AI_Mail_Assistant

AI_Mail_Assistant is an **AI-powered email automation agent** built with **n8n** and **Large Language Models (LLMs)**.  
It allows users to send emails via **natural language chat commands**, automatically fetch recipients from Google Contacts, draft professional emails, and securely send them via Gmail.

This project demonstrates the power of **AI-driven workflow automation**, replacing repetitive email tasks with a smart, secure, and scalable solution.
